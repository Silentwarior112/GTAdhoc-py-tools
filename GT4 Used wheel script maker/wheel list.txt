This contains the full list of wheel codes found in GT4 Online SCUS-97436.

Some wheels are the aftermarket ones that are already present in the wheel shop,
some are unnecessary duplicates, some are glitchy,
and some are default wheels that normally can't be accessed,
but most the default wheels are fully functional and also have icons that can be
added to the wheel shop's .gpb container, located in the piece folder.

In addition to the cars themselves, some wheels for regional exclusives are missing from GT4 Online SCUS-97436,
so those will need to be added into the script and spec database if you import those in.
A notable example is the Mini Cooper 1.3i from the Japanese version, "rov10001".
